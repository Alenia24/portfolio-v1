## Tech Stack
1. React
2. React Router
3. Bootstrap
4. EmailJS
---

## Features
- Showcases portfolio projects with images and links
- Contact form integrated with EmailJS
- Navigation with React Router
- Responsive Design

## Getting Started
### Installation
```bash
git clone https://github.com/Alenia24/portfolio-v1.git
cd portfolio-v1
npm install
npm run dev
```
